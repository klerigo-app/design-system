The EspañoLenka
Design System

A warm, playful visual language for Lenka's Spanish lessons — from the color tokens and type scale to the lesson cards, streaks and quizzes that guide every student through the app.

Foundations

Iconography

Icons are friendly and simple, matching Baloo 2's rounded feel. Use a single rounded line family — Phosphor (Rounded) is the reference set — with these rules so the app stays consistent.

Sample glyphs · 2px rounded stroke
On brand color fills, 44px min tap target.
Grid & size
Draw on a 24px grid. Ship at 20 / 24 / 28px. Keep 2px optical padding inside the box.
Stroke & corners
2px stroke, round caps and joins. Match the wordmark's soft, friendly geometry — no sharp corners.
Color
Default Ink. Use one brand hue to signal state — Coral for actions, Teal for progress, Sunshine for rewards. Never multicolor a single icon.
Components

Buttons

Baloo 2 500, radius 16, generous padding. The primary button has a soft drop-shadow "lift" that presses down on click. Buttons carry the app forward — one primary action per screen.
